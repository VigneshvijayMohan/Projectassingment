## Steps to run the app

### 1. Install `pipenv`

```bash
pip install pipenv
```



### 2. Install dependencies

```bash
pipenv install
```

### 3. Make migrations

```bash
py manage.py makemigrations
```

### 4. Migrate

```bash
py manage.py migrate
```

### 5. Run the app

```bash
py manage.py runserver
```
